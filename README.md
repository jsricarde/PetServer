# PetServer

PetServer is a backend as a service (BASS) for a Petime platform is development in node.js with express.js.

Prerequisites:
  - node.js and npm installed.



### Version
1.0.0

### Installation

PetServer requires [Node.js](https://nodejs.org/) v4+ to run.

You need npm and node.js installed globally

Download the git. repository
```sh
$ git clone git@github.com:jsricarde/PetServer.git
```
Install the node_module dependences

```sh
$ cd PetServer
$ npm install
```

### Usage

Issue the command `node app.js` in your project's directory.
